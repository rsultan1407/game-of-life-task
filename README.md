# CI/CD Pipeline using Jenkins, Docker

## Project Overview

This project demonstrates a complete CI/CD pipeline using Jenkins and Docker for a Java-based application.

---

## Steps Followed

### 1. Source Code Management

- Created GitHub repository
- Uploaded Java application

### 2. Git Operations

- Cloned repository
- Created main branch
- Pushed changes

### 3. Jenkins Setup

- Installed Jenkins
- Configured Node (Agent)
- Executed jobs on Node

### 4. Docker Setup

- Installed Docker
- Verified installation
- Provided Jenkins permissions

### 5. CI/CD Pipeline

- Checkout Code
- Build Application
- Run Tests
- Build Docker Image
- Run Container
- Verify Deployment
